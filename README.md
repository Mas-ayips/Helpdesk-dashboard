# Ayip’s Helpdesk Website

Website profesional untuk layanan Helpdesk Ayip’s — interaktif, modern, dan dibuat dengan TailwindCSS.

## 🌐 Fitur

- 🔥 Dark Mode
- 🕒 Jam Live
- 🧾 Marquee teks berjalan
- ☎️ Tombol WhatsApp
- 💻 Responsif di HP & Desktop

## 🛠️ Dibuat dengan

- TailwindCSS
- HTML5
- AOS Animation
- Lucide Icons
